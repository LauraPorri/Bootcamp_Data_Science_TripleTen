# Python básico

En este Sprint trabajé con un dataset de música con la finalidad de comparar las preferencias musicales de usuarios de 2 localidades diferentes. 

Para ello utilicé la librería Pandas, usé la función read_csv, los métodos head() e info() para observar el tipo de datos. Cambié los nombres de las columnas, también traté valores ausentes con los métodos isna, isnull, fillna, así como los duplicados con los métodos duplicated, drop_duplicates.  

También utilicé el metodo unique() para verificar si había datos únicos o repetidos, utilicé el método groupby para agrupar algunos datos de columnas y realicé filtrado de datos así como algunos métodos para sacar conclusiones a cerca de la preferencia musical de los usuarios de cada localidad.
